# Welcome to The Slow Net API Documentation

This documentation provides a comprehensive guide on how to programmatically interact with your Slow Net account and its unique features using secure API keys. Whether you're building custom integrations, automating tasks, or simply exploring, our API is designed to be developer-friendly and efficient.

The Slow Net is a calm-tech platform focused on deep thinking, digital minimalism, and reclaiming attention. Our API extends this philosophy, allowing you to manage your digital presence with precision and control.

---

## What You Can Do with the API

* **Account Management:** Retrieve and update your SlowNet display name.
* **Email Forwarding:** Check status and toggle your private email alias.
* **Discord Slowhooks:** Send and edit rich messages to your Discord servers.
* **Site Configuration:** Get and update your public SlowNet page's content, appearance, and visibility.
