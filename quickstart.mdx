# Quickstart Guide

This guide will help you get started with The Slow Net API, from obtaining your API key to making your first authenticated request.

## 1. Get Your API Key

To interact with The Slow Net API programmatically, you'll need an **API Key**.

1.  **Log in to your Dashboard:** Go to [https://theslow.net/dashboard](https://theslow.net/dashboard).
2.  **Navigate to API Settings:** Look for a section related to "API Keys" or "Developer Settings" within your dashboard.
3.  **Generate a New Key:** Follow the instructions to generate a new API Key. Your key will be in the format `UUID.SECRET_STRING`.
4.  **Keep it Secure:** Treat your API Key like a password. Do not share it publicly or commit it directly to your source code.

## 2. Understanding Authentication

All authenticated API requests require your API Key to be sent in the `X-API-Key` HTTP header.

**Example Header:**

```
X-API-Key: your_uuid.your_secret_string_here
```

## 3. API Base URLs

All API endpoints are relative to one of the following base URLs:

* **Production:** `https://theslow.net/api/`
* **Local Development:** `http://localhost:3000/api/`

## 4. Make Your First API Call

Let's make a simple `GET` request to retrieve your display name.

**Endpoint:** `GET /api/account/get-display-name`

Replace `your_uuid.your_secret_string_here` with your actual API Key.

### Using cURL (Command Line)

```bash
curl -X GET \
  [https://theslow.net/api/account/get-display-name](https://theslow.net/api/account/get-display-name) \
  -H 'X-API-Key: your_uuid.your_secret_string_here'
```

### Using JavaScript (Fetch API)

```javascript
async function getDisplayName() {
  const apiKey = 'your_uuid.your_secret_string_here'; // Replace with your actual API Key

  try {
    const response = await fetch('[https://theslow.net/api/account/get-display-name](https://theslow.net/api/account/get-display-name)', {
      method: 'GET',
      headers: {
        'X-API-Key': apiKey,
        'Content-Type': 'application/json'
      }
    });

    if (!response.ok) {
      const errorData = await response.json();
      console.error('API Error:', errorData.message);
      return;
    }

    const data = await response.json();
    console.log('Your Display Name:', data.displayName);
  } catch (error) {
    console.error('Network or other error:', error);
  }
}

getDisplayName();
```

### Expected Response (Success)

```json
{
  "displayName": "YourCurrentDisplayName"
}
```

### Expected Response (Error - Unauthorized)

If your API Key is missing or invalid, you will receive a `401 Unauthorized` response:

```json
{
  "message": "Unauthorized"
}
```
