# Email Forwarding

This section details the API endpoints for managing your SlowNet email forwarding alias (`yourname@theslow.net`).

---

## Get Email Forwarding Status

Retrieve the current email forwarding status and the destination email address for the authenticated user.

### Endpoint

`GET /api/email-forwarding/email-forwarding-status`

### Authentication

**API Key (Recommended for programmatic access):**
Provide your API key in the `X-API-Key` HTTP header.

```
X-API-Key: UUID.SECRET_STRING
```

**Session/Cookie (For dashboard-initiated requests):**
Automatically handled when logged in via the dashboard.

### Request

This endpoint requires no request body or query parameters.

### Example Request

```bash copy
curl -X GET \
  -H "X-API-Key: <YOUR_FULL_API_KEY>" \
  [https://theslow.net/api/email-forwarding/email-forwarding-status](https://theslow.net/api/email-forwarding/email-forwarding-status)
```

### Response

Returns a JSON object indicating whether forwarding is `enabled` and the `email` address it forwards to.

#### Success Response (200 OK)

```json
{
  "enabled": true,
  "email": "your-personal-email@example.com"
}
```json
{
  "enabled": false,
  "email": ""
}
```

#### Error Response (401 Unauthorized)

```json
{
  "error": "Unauthorized: You must be logged in or provide a valid API Key."
}
```

## Toggle Email Forwarding

Enable or disable email forwarding for the authenticated user's SlowNet alias. When enabling, a destination email address must be provided.

### Endpoint

`POST /api/email-forwarding/email-forwarding-toggle`

### Authentication

**API Key (Recommended for programmatic access):**
Provide your API key in the `X-API-Key` HTTP header.

```
X-API-Key: UUID.SECRET_STRING
```

**Session/Cookie (For dashboard-initiated requests):**
Automatically handled when logged in via the dashboard.

### Request Body

`Content-Type: application/json`

| Field     | Type      | Required | Description                                                                                             |
| :-------- | :-------- | :------- | :------------------------------------------------------------------------------------------------------ |
| `enabled` | `boolean` | Yes      | Set to `true` to enable forwarding, `false` to disable.                                                 |
| `email`   | `string`  | Conditional | Required if `enabled` is `true`. The email address to forward to. Ignored if `enabled` is `false`. |

### Example Request (Enable Forwarding)

```bash copy
curl -X POST \
  -H "X-API-Key: <YOUR_FULL_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
        "enabled": true,
        "email": "your-personal-email@example.com"
      }' \
  https://theslow.net/api/email-forwarding/email-forwarding-toggle
```

### Example Request (Disable Forwarding)

```bash copy
curl -X POST \
  -H "X-API-Key: <YOUR_FULL_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
        "enabled": false
      }' \
  https://theslow.net/api/email-forwarding/email-forwarding-toggle
```

### Response

Returns a success message upon successful toggling of email forwarding.

#### Success Response (200 OK)

```json
{
  "success": true
}
```
